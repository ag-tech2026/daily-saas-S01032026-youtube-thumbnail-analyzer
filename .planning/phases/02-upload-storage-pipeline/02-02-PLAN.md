**Create src/components/upload/file-upload.tsx:**

A "use client" component that provides drag-and-drop and file picker UI.

Props: `credits: number` (user's current credit balance), `onUploadComplete?: (data: { analysisId: string; imageUrl: string; credits: number }) => void`.

Implementation:
1. Use `useDropzone` from react-dropzone with config:
   - `accept: { "image/jpeg": [".jpg", ".jpeg"], "image/png": [".png"] }`
   - `maxSize: 10 * 1024 * 1024` (10MB)
   - `multiple: false` (one file at a time)
   - `onDrop` callback handles the upload flow

2. State: `uploading: boolean`, `error: string | null`, `showCreditsDialog: boolean`

3. On drop/file select:
   - If `credits <= 0`: set `showCreditsDialog = true` and return (do NOT upload)
   - If file rejected by dropzone (wrong type, too large): set appropriate error message
     - For size: "File too large. Maximum size is 10MB"
     - For type: "Only PNG and JPG images are allowed"
   - If file accepted:
     - Set `uploading = true`, clear error
     - Create `FormData`, append file as "file"
     - `fetch("/api/upload", { method: "POST", body: formData })`
     - On success (200): call `onUploadComplete` with response data, clear uploading
     - On 402: set `showCreditsDialog = true`, clear uploading
     - On 400/other error: set error from response JSON, clear uploading

4. UI layout (use Tailwind, shadcn tokens):
   - Outer container with dashed border (2px dashed), rounded-lg, padding
   - When dragging over: highlight border color (border-primary), light background tint (bg-primary/5)
   - When idle: border-muted-foreground/25
   - Center content vertically and horizontally
   - Upload icon (use `Upload` from lucide-react), size h-12 w-12, text-muted-foreground
   - Text: "Drag & drop your poker screenshot here" (text-lg font-medium)
   - Subtext: "or click to select a file" (text-sm text-muted-foreground)
   - Subtext: "PNG or JPG, max 10MB" (text-xs text-muted-foreground)
   - When uploading: show a spinner or "Uploading..." text with disabled state
   - When error: show error message in text-destructive below the drop zone
   - Render `<InsufficientCreditsDialog>` with showCreditsDialog state
